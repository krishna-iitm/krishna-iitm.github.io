TQD+SimModule - Model Documentation

This documentation presents the module level description of the simulator that evaluates performance of routing policy Tandem Queue Decomposition(TQD)

The documentation contains the code-level description of all simple,compound modules, interfaces which can be navigated on the left side bar of TQDSimModule.

Starting a top-down approach with Networks -> RandomGraph on the navigation bar helps understand the whole structure in a faster manner.

This is not a user manual to the simulator rather a document that helps peers with prior knowledge on the OMNeT++ framework to adapt the network controller for other applicable scenarios.

We request the users to cite our paper "Fast and Secure Routing Algorithms for Quantum Key Distribution Networks"
